# Hotel-Booking-Cancellation-Prediction

## Overview
This project predicts whether a hotel booking will be canceled using machine learning classification models. The objective was to maximize ROC/AUC performance in a Kaggle-style prediction challenge.

## Tools Used
R, caret, recipes, tidyverse, gbm, ranger, glmnet, cross-validation, ROC/AUC

## Models Compared
- Logistic Regression
- Decision Tree
- Random Forest
- Gradient Boosted Trees
- KNN

## Key Methods
- Data cleaning and preprocessing
- Missing value handling
- Feature engineering
- Cross-validation
- Hyperparameter tuning
- Model evaluation using ROC/AUC

## Final Result
Gradient boosted trees achieved the strongest individual model performance with 0.8765 cross-validated ROC/AUC.

## Reports
- Technical report: includes code, modeling process, feature engineering, and evaluation
- Non-technical report: summarizes findings for hospitality stakeholders
